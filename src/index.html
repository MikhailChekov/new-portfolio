React table app

Приложение для отображения данных пользователей в виде таблицы, создавалось в соответствии с ТЗ, во время задания старался использовать дополнительные библиотеки по минимуму, писать все на чистом js, для того чтобы уменьшить вес, и для изучения/закрепления знаний языка.


Функционал:

  • Загрузка данных через Api, вывод нужного количества данных, в зависимости от запроса.
  • Возможность сортировки/поиска/добавления и вывода данных конкретного пользователя.
  • Фильтрация значений формы.

Технологии и библиотеки, используемые при создании:

  • React, React (Hooks), React-bootstrap, react-paginate.
  • Регулярные выражения для валидации формы.
